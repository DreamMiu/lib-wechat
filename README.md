## lib-wechat

1. nodejs端的lib-wechat 实现
2. 按照icqq的规范，实现了的部分接口

## 使用样例

1. 安装依赖：

```shell
npm install lib-wechat --save
```

2. 引入：

```javascript
const {Client} = require('lib-wechat');
const client = new Client();
client.login()
client.on('message', (e) => {
    console.log(e);
    if (e.message === 'hello') {
        e.reply('world');
    }
})
```
